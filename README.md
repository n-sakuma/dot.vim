# セットアップ
---------------

# 事前準備

もし、すでに .vimrc, ~/.vim がある場合、削除するかmvしてbackupをとっておく

> rm -rf ~/.vim  もしくは mv ~/.vim{,.old}

# リポジトリを取得(clone)

> git clone git@github.com:n-sakuma/dot.vim.git ~/.vim

# セットアップ

> ~/.vim/setup.sh

# Plugin の設定 - vundle管理のplugin をインストール

## インストール

> :NeoBundleInstall


## 一覧

> :NeoBundleList
